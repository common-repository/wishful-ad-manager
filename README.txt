== Description ==

Wishful Ads manager is a totally customized shortcode driven easy to manage plugin used for custom banner ads, scripts ads, Google Adsense.


== ADVERTISEMENT  TYPES ==

Wishful Ads manager supports Banner Ads and Custom Script code. 


== SHORTCODE SUPPORT == 

You can display an advertisement using shortcode anywhere on your website.
